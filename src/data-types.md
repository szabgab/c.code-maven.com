# Data Types in C


| name    | description |
| ------- | ----------- |
| `char`  | character - a single byte        (between -128  and 127)  |
| `int`   | integer  (16 bit, 32 bit?)         |
| `short` | short integer                      |
| `long`  | long integer                       |
| `float` |  floating point (typically 32 bit) |
| `double` | double-precision floating point   |
